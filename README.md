This project is a simple **web interface** built using **HTML, CSS, and JavaScript**.  
It demonstrates how to create:  
- A **Side Navigation Menu** for easy navigation  
- A **Search Categories Area** for browsing items by type  
- An **Ingredients Section** to display details of selected items  
- A **Contact Us Page** with a simple form
